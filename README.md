# AI Magic Sort Solver 🧪

An advanced, interactive web application designed to solve complex Magic Sort puzzles. Built with Next.js, React, and Framer Motion, this tool provides players with an intuitive interface to recreate their in-game puzzle boards and find optimal, step-by-step solutions.

🔗 **Live Website:** [aimagicsortsolver.com](https://aimagicsortsolver.com)

## ✨ Key Features
- **Visual Board Editor:** Easily set up complex puzzle states with custom colors, tube sizes, and hidden layers.
- **Support for Special Objects:** Fully handles advanced puzzle mechanics including Keys, Locks, Ice, Boxes, Labels, and Curtains.
- **Smart AI Solver:** Instantly calculates the most efficient move path to clear the board without getting stuck.
- **Game-Like Playback:** Follow along with smooth, liquid-pouring animations that mimic the actual game.
- **Responsive Design:** A premium, mobile-friendly interface with rich aesthetics, smooth micro-animations, and a cohesive design system.

## 🚀 Tech Stack
- **Framework:** Next.js (App Router) & React
- **Styling:** Vanilla CSS with custom design tokens for a premium glassmorphism aesthetic
- **Animations:** Framer Motion (for fluid, event-driven pouring animations)
- **Deployment:** Optimized for Vercel / edge environments
